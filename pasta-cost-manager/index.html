Multi-Device Online Cloud Sync

To sync your recipe costs, worker hours, and expenses in real-time across your phone, tablet, and other devices, you can set up a private cloud bin.

How do I set this up for free in 30 seconds?
  1. Go to JSONBin.io and register for a free account.
  2. In your JSONBin dashboard, copy your **API Key** (from the API Keys section).
  3. Paste your API Key below and click **"Create Private Cloud Sync Bin"**.
  4. The app will generate a secure, private **Sync Code**. Copy this code and paste it along with your API Key into the app settings on your phone or other devices!
Cloud Sync is currently disabled. All changes are being saved locally on this device.
